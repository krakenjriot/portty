Basic Syntax 
============
Example. Arduino Uno connected to com10, b1 as board name (any name is possible)
Syntax     : portty.exe <board-name> <com-port>
Example    : portty.exe myboard com5
  
`To start the portty.exe with the above setup`

```sh
portty.exe b1 com10
```
